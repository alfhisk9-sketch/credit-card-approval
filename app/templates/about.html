Project Overview

AI-Based Credit Card Approval Prediction System is a machine learning web application developed as part of the SmartBridge Internship Program. The system predicts whether a credit card application is likely to be approved by analyzing customer financial and personal information using multiple machine learning algorithms. The application provides real-time predictions, confidence scores, risk analysis, and detailed visual analytics through a responsive Flask-based web interface.. The best-performing model (XGBoost) is deployed on IBM Watson Machine Learning for real-time inference.

The system implements a complete end-to-end ML pipeline — from data preprocessing and model training to cloud deployment and a production-grade web interface built with Flask and Bootstrap 5.
